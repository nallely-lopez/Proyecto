Automatisation de l'installation de produits NI

Vous pouvez automatiser l'installation de la plupart des produits NI en utilisant des arguments de ligne de commande pour supprimer certaines (voire toutes) des interfaces utilisateur et boîtes de dialogue. Cependant, à partir des versions d'août 2012 (les produits utilisant la version 3.1 des installeurs NI ou une version supérieure), il se peut que vous deviez effectuer des étapes supplémentaires avant ou pendant les installations silencieuses de logiciels NI.

Si le produit NI que vous installez utilise Microsoft .NET 4.0, il est possible que l'installeur .NET s'exécute avant l'installation des logiciels NI et que vous deviez redémarrer pour que l'installation des logiciels NI puisse commencer. Pour éviter le redémarrage requis par .NET, installez .NET 4.0 avant d'installer des logiciels NI.

Pour obtenir des informations complémentaires sur l'automatisation de l'installation de produits NI, reportez-vous aux articles suivants de la Base de connaissances :

  • Pour en savoir plus sur les installations silencieuses de produits NI individuels, reportez-vous à l'article KB 4CJDP38M, Automatisation de l'installation d'un seul installeur.
  • Pour en savoir plus sur les installations silencieuses des produits d'une suite NI, telle que NI Developer Suite, reportez-vous à l'article KB 4GGGDQH0, Automatisation de l'installation d'un installeur de suite.
  • Pour déterminer quelle version des installeurs NI votre produit contient, reportez-vous à l'article KB 4CJDR18M, Comment puis-je déterminer le type et la version de mon installeur National Instruments ?.

Comment utiliser les logiciels NI sous Microsoft Windows 8.x

Lorsque vous installez des logiciels National Instruments sous Microsoft Windows 8.x, vous remarquerez quelques vignettes supplémentaires sur l'affichage Applications, y compris des raccourcis vers les logiciels d'applications NI, tels que NI LabVIEW, Measurement & Automation Explorer (MAX) et le nouveau Lanceur d'applications NI.

Utilisation du Lanceur d'applications NI

Le Lanceur d'applications NI vous permet de trouver facilement les produits NI installés et de les lancer. Il vous permet de trouver les produits NI de la même façon qu'avec le menu Démarrer des versions antérieures de Microsoft Windows. Pour utiliser le Lanceur d'applications NI, cliquez sur la vignette Lanceur d'applications NI de l'affichage Applications. Le Lanceur d'applications NI lance le bureau et ouvre un menu qui contient la liste des produits NI. Cliquez sur un produit NI pour le lancer.

Remarque  NI MAX n'apparaît pas sous forme de raccourci sur le bureau lorsqu'il est installé sous Microsoft Windows 8.x. Vous pouvez lancer NI MAX soit à partir de l'affichage Applications, soit à partir du Lanceur d'applications NI.

Épinglage des applications fréquemment utilisées

Pour plus de facilité, vous pouvez épingler les applications NI que vous utilisez le plus souvent (MAX, par exemple) sur l'écran d'accueil ou sur la barre des tâches du bureau, selon vos besoins.

Recherche de programmes

Pour trouver une application spécifique ou des fichiers associés, comme des fichiers de documentation, depuis l'écran d'accueil, vous pouvez accéder à tous les fichiers installés à partir de l'affichage Applications. Pour accéder à l'affichage Applications, cliquez avec le bouton droit n'importe où sur l'écran d'accueil et sélectionnez Toutes les applications (Windows 8) ou cliquez sur la flèche située dans l'angle inférieur gauche de l'écran (Windows 8.1).

Pour en savoir plus sur le support NI de Windows 8.x, allez sur ni.com/windows8.

NI arrêtera le support de Microsoft Windows Vista, Windows XP et Windows Server 2003 en 2016

La plupart des produits National Instruments ne supporteront plus Microsoft Windows Vista, Windows XP et Windows Server 2003 dès le premier juillet 2016. La plupart des versions des produits NI expédiées après le premier juillet 2016 ne s'installeront pas et ne s'exécuteront pas sous Windows Vista, Windows XP ni Windows Server 2003.
