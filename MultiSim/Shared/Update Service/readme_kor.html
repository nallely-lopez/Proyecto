NI 업데이트 서비스 Readme

NI 업데이트 서비스는 사용자 컴퓨터에 NI 소프트웨어 및 드라이버의 업데이트가 필요한지 확인하고 업데이트를 수행합니다. 이 파일에는 NI 업데이트 서비스에서 알려진 유의사항 및 수정된 버그의 일부 리스트를 포함한 중요한 최신 정보가 있습니다.

제품 보안 및 중요한 업데이트

ni.com/security 페이지에서 National Instruments 제품에 대한 보안 알림사항을 보고 구독할 수 있습니다. National Instruments의 중요한 업데이트에 대한 정보는 ni.com/critical-updates를 방문하여 확인하십시오.

알려진 유의사항 및 버그 수정

온라인에서 소프트웨어 및 문서의 알려진 유의사항을 찾을 수 있습니다. NI 업데이트 서비스의 알려진 유의사항이 정리된 최신 목록을 확인하려면 ni.com/info에서 정보 코드 UpdateServiceKnownIssues를 입력하십시오. 버그 수정이 정리된 최신 목록을 확인하려면 ni.com/info에서 정보 코드 UpdateServiceFixList를 입력하십시오.

NI 제품 자동 설치하기

명령 라인 인수를 사용하여 설치 프로그램의 사용자 인터페이스 및 대화 상자를 모두 또는 일부 비활성화하면 대부분 NI 제품의 설치를 자동화할 수 있습니다. 그러나, 2012년 8월 출시부터 (NI 설치 프로그램 버전 3.1 또는 이후 버전을 사용하는 제품의 경우), NI 소프트웨어의 자동 설치 이전 또는 도중 추가로 수행해야 할 단계가 있을 수 있습니다.

설치하는 NI 제품이 Microsoft .NET 4.0을 사용하는 경우, NI 소프트웨어가 설치되기 전에 .NET 설치 프로그램이 실행되고 NI 소프트웨어 설치가 시작되기 전에 재부팅이 필요할 수 있습니다. .NET 재부팅을 피하려면, NI 소프트웨어를 설치하기 전에 별도로 .NET 4.0을 설치하십시오.

NI 제품의 자동 설치에 대한 더 자세한 정보는 다음 기술지원 데이터베이스 문서를 참조하십시오:

  • 개별 NI 제품의 자동 설치에 대한 더 자세한 정보는 KB 4CJDP38M, Automating the Installation of a Single Installer를 참조하십시오.
  • NI Developer Suite과 같이 세트로 묶인 NI 제품의 자동 설치에 대한 더 자세한 정보는 KB 4GGGDQH0, Automating the Installation of a Suited Installer를 참조하십시오.
  • 제품에 포함된 NI 설치 프로그램의 버전을 확인하려면 KB 4CJDR18M, How Can I Determine the Type and Version of My National Instruments Installer?를 참조하십시오.

NI 소프트웨어를 Microsoft Windows 8.x에서 사용하는 방법

Microsoft Windows 8.x에서 National Instruments 소프트웨어를 설치하면, 앱 보기에 NI LabVIEW, Measurement & Automation Explorer (NI MAX) 및 NI 시작 관리자와 같은 NI 어플리케이션 소프트웨어 제품의 바로 가기를 포함하여 몇 가지 타일이 추가되는 것을 확인할 수 있습니다.

NI 시작 관리자 사용하기

NI 시작 관리자를 사용하여 컴퓨터에 설치된 NI 제품을 찾고 시작할 수 있습니다. 이 어플리케이션은 Microsoft Windows 이전 버전의 시작 메뉴와 비슷하며, 이를 사용하여 NI 제품을 찾을 수 있습니다. NI 시작 관리자를 사용하려면, 앱 보기에서 NI 시작 관리자 타일을 클릭하십시오. NI 시작 관리자는 데스크톱을 열고 NI 제품 리스트가 있는 메뉴를 엽니다. NI 제품을 클릭하여 시작합니다.

노트  NI MAX는 Windows 8.x에 설치할 경우 데스크톱 바로 가기로 나타나지 않습니다. 앱 보기 또는 NI 시작 관리자 내에서 NI MAX를 시작할 수 있습니다.

자주 사용하는 어플리케이션 고정하기

사용자의 편의를 위해 MAX와 같이 가장 자주 사용하는 NI 어플리케이션을 시작 화면 또는 테스크톱의 작업 표시줄에 고정할 수 있습니다:

현재 어플리케이션아이콘 위치고정 방법
Windows 8.x 데스크톱데스크톱 작업 표시줄어플리케이션에서 마우스 오른쪽 버튼을 클릭한 후 작업 표시줄에 고정 선택
Windows 8.x 데스크톱 시작 화면어플리케이션에서 마우스 오른쪽 버튼을 클릭한 후 시작 화면에 고정 선택
Windows 8.x 시작 화면데스크톱 작업 표시줄어플리케이션에서 마우스 오른쪽 버튼을 클릭한 후 화면 아래쪽의 메뉴 모음에서 작업 표시줄에 고정 선택

모든 프로그램 찾기

시작 화면에서 특정 어플리케이션 또는 문서 파일과 같은 관련 파일을 찾으려는 경우, 앱 보기에서 설치된 모든 파일을 찾을 수 있습니다. 앱 보기로 가려면, 시작 화면의 아무 곳에서나 마우스 오른쪽 버튼을 클릭한 후 앱 모두 보기를 선택하거나(Windows 8) 화면의 왼쪽 하단에 있는 아래 방향 화살표를 클릭합니다(Windows 8.1).

NI의 Windows 8 x 지원에 대한 더 자세한 정보를 알아보려면 ni.com/windows8을 방문하십시오.

2016년 NI의 Microsoft Windows Vista, Windows XP 및 Windows Server 2003에 대한 지원 중단

2016년 7월 1일부터 대부분의 National Instruments 제품은 Microsoft Windows Vista, Windows XP 및 Windows Server 2003에 대한 지원을 중단합니다. 2016년 7월 1일 이후 출시되는 NI 제품의 대부분 버전은 Windows Vista, Windows XP 또는 Windows Server 2003에서 설치 또는 실행할 수 없습니다. NI 어플리케이션 소프트웨어 제품 수명에 대한 상세 정보는 ni.com/info에서 다음의 정보 코드를 입력하십시오:
