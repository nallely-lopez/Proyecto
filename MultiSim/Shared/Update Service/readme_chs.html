自动安装NI产品的方法

大多数NI产品都可通过命令行自动安装，通过设置命令行参数完成安装程序用户界面或对话框的设置。不过从2012年8月发行的版本（NI Installers 3.1或更高版本）开始，实现无人值守的NI软件安装时需要作一些额外的设置。

如待安装的NI产品依赖于Microsoft .NET 4.0，则在启动NI软件安装之前，.NET安装程序会首先启动并在完成后提示重启。如需避免上述情况，可在安装NI软件之前单独安装.NET 4.0。

关于自动安装NI产品的详细信息，见下列知识库文章：

  • 关于在无人值守的情况下安装NI产品的详细信息，见KB 4CJDP38M（Automating the Installation of a Single Installer）。
  • 关于在无人值守的情况下安装NI产品套件的详细信息，见KB 4GGGDQH0（Automating the Installation of a Suited Installer）。
  • 如需确定当前产品NI Installer的版本，见KB 4CJDR18M（How Can I Determine the Type and Version of My National Instruments Installer?）。

在Microsoft Windows 8.x上使用NI软件

在Microsoft Windows 8.x平台安装NI软件时，应用视图上会增加一些图块，作为指向各个NI应用程序（例如，NI LabVIEW、Measurement & Automation Explorer－NI MAX和NI启动器）的快捷方式。

使用NI启动器

“NI启动器”可用于定位和启动计算机上已安装NI产品。其功能类似于Microsoft Windows早期版本的“开始”菜单。如需使用“NI启动器”，可在应用视图中单击NI启动器图块。“NI启动器”将启动桌面并打开一个菜单，其中包含一系列NI产品。单击产品名称可启动NI产品。

注：  在Microsoft Windows 8.x平台中，NI MAX不会被添加桌面快捷方式。可直接通过应用视图或NI启动器打开NI MAX。

将常用的应用程序固定到开始屏幕

您可将常用的NI应用程序（例如，MAX）固定到“开始”屏幕或桌面任务栏：

当前应用程序环境图标位置固定应用
Windows 8.x桌面桌面任务栏右键单击应用程序并选择固定到任务栏
Windows 8.x桌面 开始屏幕右键单击应用程序并选择固定到“开始”屏幕
Windows 8.x开始屏幕桌面任务栏右键单击应用程序并从屏幕底部的菜单栏选择固定到任务栏

查找所有程序

如需在“开始”屏幕中查找特定的应用程序或相关文件（例如，文档），右键单击开始屏幕的任意位置，选择所有应用 (Windows 8)，或在屏幕左下方单击向下箭头(Windows 8.1)，在应用视图中查看全部已安装文件。

关于Windows 8.x平台的NI支持信息，请访问ni.com/windows8。

2016年起NI将不再支持Microsoft Windows Vista，Windows XP和Windows Server 2003
