Readme zum NI-Lizenzmanager

Mit Hilfe des Lizenzmanagers von National Instruments können Sie Ihre Lizenzen aktivieren und verwalten oder NI-Produkte auf Ihrem System für die Zuweisung von Lizenzen durch Volumenlizenzserver konfigurieren.

Bekannte Probleme

In der folgenden Tabelle werden alle bekannten Probleme im NI-Lizenzmanager 3.7 mit Hinweisen zur Problemlösung aufgeführt.

IDKnown Issue
136355, 136356Computers using older versions of NI software, such as NI TestStand 3.0, NI LabVIEW 7.1, or NI LabWindows™/CVI™ 7.0 cannot check out a license from a volume license server.

Workaround: Contact NI technical support at support@ni.com for assistance in resolving this issue. Refer to KnowledgeBase article 475FE2QA for more information.

Behobene Fehler

Nachfolgend wird ein Teil der Probleme beschrieben, die im NI-Lizenzmanager 3.7 behoben wurden. Um den Rahmen dieser Liste nicht zu sprengen, wurden nur die wichtigsten Probleme aufgeführt.

ID Fixed Issue
220216Fixed an issue where disconnected licenses were not accounted for in NI License Manager in-use count.

Note:  Requires NI Volume License Manager 3.0 or later.
306393Fixed an issue where a Try Again/Continue/Cancel Media prompt appeared during activation if the media is changed.

Automatische Installation von NI-Produkten

Die Installation der meisten Produkte von National Instruments kann über Kommandozeilenargumente gestartet werden, so dass keine (oder nicht alle) Dialogfelder des Installationsprogramms angezeigt werden. Ab August 2012 veröffentlichte NI-Softwareprodukte (die mit einer NI-Installer-Version ab einschließlich 3.1 arbeiten) erfordern jedoch möglicherweise zusätzliche Schritte vor oder während der automatischen Installation.

Wenn die zu installierende NI-Software mit Microsoft .NET 4.0 arbeitet, wird das Installationsprogramm von .NET vor dem der NI-Software ausgeführt und erfordert möglicherweise zwischendurch einen Neustart. Wenn Sie den Neustart verhindern möchten, müssen Sie Microsoft .NET 4.0 vorher separat installieren.

Weitere Informationen finden Sie in folgenden KnowledgeBase-Artikeln:

  • Die automatische Installation einzelner NI-Produkte wird in der KB 4CJDP38M, Automating the Installation of a Single Installer, beschrieben.
  • Die automatische Installation von NI-Produkten innerhalb einer Suite (z. B. der NI Developer Suite) wird in der KB 4GGGDQH0, Automating the Installation of a Suited Installer, beschrieben.
  • Die Version des NI-Installers, auf dem das Installationsprogramm Ihres Produkts aufbaut, ist in der KB 4CJDR18M, How Can I Determine the Type and Version of My National Instruments Installer?, beschrieben.
