NI许可证管理器自述文件

NI许可证管理器用于激活并管理许可证文件、配置系统上安装的NI产品，以及从批量许可证服务器上获取许可证。

自动安装NI产品

可使用命令行参数来自动安装大多数NI产品，以减少安装程序用户界面和对话框交互。 从2012年8月开始（使用NI Installers 3.1及更高版本），需额外步骤才能进行NI产品后台安装。

如安装的的产品使用了Microsoft .NET Framework 4.0，.NET Framework 4.0的安装程序会在NI软件安装之前运行。开始安装NI软件之前可能需要重启计算机。 为了避免重启计算机，建议在安装NI软件之前单独安装.NET Framework 4.0。

详细信息请参考下列知识库文章：

  • 关于分别在后台单独安装NI产品的详细信息，请参考KB 4CJDP38M：在后台安装单个安装程序。
  • 关于分别在后台单独安装NI软件套件的详细信息，请参考KB 4GGGDQH0：在后台安装软件套件的安装程序。
  • 要查看产品的NI Installers版本号，请参考KB 4CJDR18M：如何判断NI安装程序的版本？
